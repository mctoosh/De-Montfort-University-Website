Course Modules

First Year

  • Business Technology – In this module you will learn about how the hardware and software work. You will also learn about how spreadsheets work, how to write a professional report, and how to make a professional website.
  • Information Systems Development – In this module, you will learn about the development of life cycle. You will also learn about working together with a group, the user requirements specifications, designing the user requirement specification, and making a final prototype.
  • Database Management and Reporting – In this module, you will learn about designing and understanding database structure design. You will be designing a working database on a visual studio using C# language, you will be working independently and also as a team.
  • Visual Web Development – In this module you will learn about the fundamentals of programing with C#. You will be designing a working website using C# language and little bit of HTML and CSS as well. .

Second Year

  • Project Management and Development – In this module, you will learn about designing and building a system that is professional from start to final implementation.
  • Interactive Systems Design and Evaluation – In this module, you will be taught and made to understand what makes a system usable.
  • Multimedia and Internet Technology – In this module, you will be making a web based multi-media by exploring the software, for example, video, text, graphics animation.
  • Advanced Programming in Visual Studio – In this module, you will learn about advanced C# programming. You will continue your C# skills that you’ve learned the previous year and use it to develop a professional website.

Third Year

  • Computing Project (Mandatory) – This would be your big final year assignment which is mandatory to complete. You will be designing and making a software that works.
  • Secure Web Application Development – This is an optional module. You will learn about how a website could be designed and applied so much so that it would be very difficult for an unauthorised access to occur.
  • Telematics – This is an optional module. You will learn and understand how a distributed telematics systems are designed and applied.
  • Multi-Service Networks – This is an optional module. You will learn about the problems and solutions that are found in modern systems in full details.
  • Fuzzy Logic and Knowledge Based Systems – This is an optional module. You will learn about the methods for modelling imprecisions and doubts that appear in a variety of applications for example washing machines.
  • Systems Building Methods and Management – This is an optional module. In this module, you will learn about two key parts of information systems development. The first one is the selection and evaluation of methodologies and the second one is the management of the Systems Development process.
  • Information Systems Strategy and Service – This is an optional module. In this module, you will learn about the information systems where the main role for the IS department is to give a good service and support for the information technology function.
  • Privacy and Data Protection – This is an optional module. In this module, you will learn about the balance of maintaining effectiveness, legal compliance and professional practice in the IT field.
  • Data Mining– This is an optional module. In this module, you will learn about the techniques you need to uncover key information from large data sets.
  • Computing Ethics – This is an optional module. In this module, you will be given the chance to search for the concepts of ethics in the context of IT, Software Engineering and other related fields.
